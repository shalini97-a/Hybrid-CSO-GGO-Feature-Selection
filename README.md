# Hybrid CSO-GGO Feature Selection for Heart Disease Prediction

## Project Overview
This project presents a hybrid feature selection approach using Cuckoo Search Optimization (CSO) and Greylag Goose Optimization (GGO) to improve heart disease prediction performance.

## Dataset
- UCI Heart Disease Dataset
- Data preprocessing and cleaning performed before training

## Technologies Used
- Python
- Google Colab
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Gradient Boosting

## Methodology
1. Data Preprocessing
2. Feature Selection using CSO
3. Feature Selection using GGO
4. Hybrid CSO-GGO Optimization
5. Model Training
6. Performance Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results
The hybrid feature selection method improved model performance by selecting the most relevant features for heart disease prediction.
